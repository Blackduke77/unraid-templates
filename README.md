# unRaid-Docker-Templates

Template files for various Docker Containers for use with unRaid Server

The docker containers referenced in this repo point to Docker Containers maintained by other individuals or maintained by me where I have added functionality.  These templates were created to make it easy for the average unRaid user to quickly install the containers without researching the various container variables and requirements.
